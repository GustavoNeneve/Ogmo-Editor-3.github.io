# [Introdução](#introduction)

O OGMO Editor é um editor de níveis 2D gratuito e de código aberto, criado por desenvolvedores indie para desenvolvedores indie. O OGMO Editor foca em um fluxo de trabalho baseado em projetos, rápido e intuitivo, para tornar o design de níveis tão eficiente e divertido quanto possível. Crie níveis usando tilesets, adicione funcionalidade posicionando entidades, decore o cenário com decals e utilize metadados personalizados para acelerar a criação de conteúdo para o seu jogo!

O OGMO Editor busca ser independente de plataforma e engine, por isso salva arquivos de projeto e de nível em formato JSON, legível por humanos e fácil de implementar. Isso facilita a integração do OGMO Editor ao seu fluxo de trabalho.

NOTA: Este manual ainda está em desenvolvimento, portanto algumas informações sobre funcionalidades existentes podem ainda não estar disponíveis.

## [Sobre o OGMO Editor](#about-ogmo-editor)

O OGMO Editor foi originalmente criado em Flash por Maddy Thorson, criadora de Towerfall. Uma segunda versão foi desenvolvida como uma aplicação nativa para Windows. O OGMO Editor 3 foi originalmente criado em Typescript por Maddy Thorson e Noel Berry, e foi utilizado na criação de Celeste. O logotipo do OGMO Editor e os ícones da aplicação foram criados por Kyle Pulver.

Esta edição comunitária do OGMO Editor 3 foi traduzida para Haxe e expandida por Caleb Cornett, Will Blanton e Austin East.
