# Decal Layers

## Ferramentas

### Decal Select Tool - <img src="https://raw.githubusercontent.com/Ogmo-Editor-3/ogmo-editor-3.github.io/gh-pages/img/icons/selection.png" class="down-eight" width="32"/>

**Botão Esquerdo do Mouse Pressionado**

- Se o cursor estiver sobre um decal, deseleciona os decals selecionados e seleciona o decal.
  + Shift: Adiciona o decal aos decals selecionados.
  + Ctrl: Alterna o estado de seleção do decal.

**Botão Esquerdo do Mouse Pressionado + Arrastar**

- Se o cursor não estiver sobre um decal, cria um retângulo de seleção.
  + Soltar Botão Esquerdo: Seleciona os decals dentro do retângulo de seleção.
  + Shift + Soltar Botão Esquerdo: Alterna o estado de seleção dos decals dentro do retângulo de seleção.

- Se o cursor estiver sobre um decal, move o(s) decal(s) selecionado(s), alinhado(s) à grade.
  + Ctrl: move o(s) decal(s) selecionado(s) sem alinhar à grade.

**Botão Direito do Mouse Pressionado + Arrastar**

- Cria um retângulo de seleção para exclusão.
  + Soltar Botão Direito: Exclui os decals dentro do retângulo de seleção.

**Atalhos**

- H: Espelha o(s) decal(s) horizontalmente.
- V: Espelha o(s) decal(s) verticalmente.
- B: Move o(s) decal(s) para o fundo.
- F: Move o(s) decal(s) para a frente.
- DELETE: Exclui o(s) decal(s).
- Ctrl + A: Seleciona todos os decals.
- Ctrl + C: Copia o(s) decal(s) selecionado(s).
- Ctrl + X: Recorta o(s) decal(s) selecionado(s).
- Ctrl + V: Cola o(s) decal(s) selecionado(s).
- Ctrl + D: Duplica o(s) decal(s) selecionado(s).
- Alt: Muda para a Decal Create Tool.

### Decal Create Tool - <img src="https://raw.githubusercontent.com/Ogmo-Editor-3/ogmo-editor-3.github.io/gh-pages/img/icons/entity-create.png" class="down-eight" width="32"/>

**Botão Esquerdo do Mouse Pressionado**

- Cria um decal a partir do modelo do pincel, alinhado à grade. Seleciona o decal automaticamente.
  + Ctrl: Cria o decal sem alinhar à grade.
  + Shift: Adiciona o decal à seleção existente.
  + Arrastar: Move o decal criado.

**Soltar Botão Esquerdo**

- Se um decal foi criado, a ferramenta é automaticamente alterada para a Decal Select Tool.

**Botão Direito do Mouse Pressionado**

- Exclui o decal na posição do cursor.
  + Arrastar: Exclui decals.

**Arrastar**

- Pré-visualiza o decal na posição do cursor.

**Atalhos**

- H: Espelha o pincel horizontalmente.
- V: Espelha o pincel verticalmente.
