# Tile Layers

## Ferramentas

### Tile Pencil Tool - <img src="https://raw.githubusercontent.com/Ogmo-Editor-3/ogmo-editor-3.github.io/gh-pages/img/icon-pencil.png" class="inverted down-eight" width="32"/>

**Botão Esquerdo do Mouse Pressionado**

- Começa a desenhar.
  + Ctrl: desenha 1x1 com pincel aleatório.
  + Soltar Botão Esquerdo: Para de desenhar.

**Botão Direito do Mouse Pressionado**

- Começa a apagar.
  + Soltar Botão Direito: Para de apagar.

**Atalhos**

- Alt: Muda para a Tile Eyedropper Tool.
- Shift: Muda para a Tile Rectangle Tool.

### Tile Rectangle Tool - <img src="https://raw.githubusercontent.com/Ogmo-Editor-3/ogmo-editor-3.github.io/gh-pages/img/icon-square.png" class="inverted down-eight" width="32"/>

**Botão Esquerdo do Mouse Pressionado + Arrastar**

- Pré-visualiza o retângulo do pincel.
  + Soltar Botão Esquerdo: Aplica o retângulo.
  + Ctrl: Aleatoriza o pincel.

**Botão Direito do Mouse Pressionado + Arrastar**

- Pré-visualiza o retângulo de exclusão.
  + Soltar Botão Direito: Exclui os tiles selecionados.

**Atalhos**

- Alt: Muda para a Tile Eyedropper Tool.
- Shift: Muda para a Tile Pencil Tool.

### Tile Line Tool - <img src="https://raw.githubusercontent.com/Ogmo-Editor-3/ogmo-editor-3.github.io/gh-pages/img/icon-line.png" class="inverted down-eight" width="32"/>

**Botão Esquerdo do Mouse Pressionado + Arrastar**

- Pré-visualiza a linha do pincel.
  + Soltar Botão Esquerdo: Aplica a linha.
  + Ctrl: Aleatoriza o pincel.

**Botão Direito do Mouse Pressionado + Arrastar**

- Pré-visualiza a linha de exclusão.
  + Soltar Botão Direito: Exclui os tiles selecionados.

**Atalhos**

- Alt: Muda para a Tile Eyedropper Tool.
- Shift: Muda para a Tile Pencil Tool.

### Tile Fill Tool - <img src="https://raw.githubusercontent.com/Ogmo-Editor-3/ogmo-editor-3.github.io/gh-pages/img/icon-floodfill.png" class="inverted down-eight" width="32"/>

**Botão Esquerdo do Mouse Pressionado**

- Preenche usando o pincel.
  + Ctrl: Preenche usando o pincel aleatório.

**Botão Direito do Mouse Pressionado**

- Preenche com a borracha.

**Atalhos**

- Alt: Muda para a Tile Eyedropper Tool.
- Shift: Muda para a Tile Pencil Tool.

### Tile Eyedropper Tool - <img src="https://raw.githubusercontent.com/Ogmo-Editor-3/ogmo-editor-3.github.io/gh-pages/img/icon-eyedropper.png" class="inverted down-eight" width="32"/>

**Botão Esquerdo do Mouse Pressionado + Arrastar**

- Seleciona tile(s) para usar como pincel.
  + Soltar Botão Esquerdo: Cria o pincel a partir dos tiles selecionados.

**Atalhos**

- Shift: Muda para a Tile Pencil Tool.
